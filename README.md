MICA é uma plataforma digital que oferece apoio, informações confiáveis e suporte emocional para pacientes com câncer. Através de um chatbot interativo, é possível tirar dúvidas, receber orientações e acessar conteúdo educativo de forma rápida e acessível.

> fornece ajuda emocional e assitencia geral (focados calro em informaçoes sobre o cançer)


- Projeto densenvolvido durande o **SENACON 2025**

## Status
- Projeto legado / versão inicial — traz funcionalidades básicas 

Tecnologias principais
- PHP, HTML, CSS, JavaScript, MYSQL
- Banco de dados relacional (MySQL) para armazenamento e consuta.


Instalação
1. Clone o repositório:
   git clone https://github.com/HeitorASM/MICApublic.git

> Atente-se ao fato que muitas funcionalidades não funcionão por 1 falta de chave de API (chatbot) 2 falta de banco de dados (login cadastro e perfil) todas retiradas por motivos óbvios
